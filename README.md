# Anonymous Repository for ARR Submission

This repository contains the code implementation for the experiments reported in the anonymous ARR submission.

The repository is anonymized for double-blind review. All identifying information will be released after acceptance.

---

## 1. Environment

### Hardware
- Default: **GPU (CUDA)**
- Automatic CPU fallback is supported if CUDA is unavailable (CPU execution is significantly slower).

### Software
- Python >= 3.8
- PyTorch >= 1.12
- CUDA >= 11.0 (optional, for GPU)

### Required Python packages
```bash
pip install -r requirements.txt
